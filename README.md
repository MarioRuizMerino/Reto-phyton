# Reto-phyton
Videojuego de RETO propuesto por Moisés Martínez Gutiérrez

El único requisito para poder disfrutar de una buena experiencia es no pegarte mucho a las paredes, ya que se queda pillado el personaje y cuesta salir.  
En caso que pase eso, en el menú voy a poner un botón que te llevará al spawn.

El videojuego lo voy a realizar en solitario.  
Todo lo que ves en el juego lo he creado yo, incluyendo los modelos el código y la idea.  
Me está llevnado mucho esfuerzo, y puedes comprobar que lo estoy haciendo yo todo artesanalmente viendo los diferentes commits y viendo el avance progresivo del juego.
